# 🧬 Algoritma Genetika (Genetic Algorithm)

## 📌 Deskripsi
Algoritma Genetika adalah salah satu metode optimasi dan pencarian berbasis *evolutionary computation* yang terinspirasi dari proses evolusi biologis seperti seleksi alam, crossover (perkawinan), dan mutasi.

:contentReference[oaicite:0]{index=0} digunakan untuk menyelesaikan masalah kompleks yang sulit diselesaikan dengan metode konvensional.

Algoritma ini bekerja dengan cara mengembangkan sekumpulan solusi (disebut *populasi*) secara bertahap hingga mendapatkan solusi terbaik.

---

## ⚙️ Konsep Dasar

Algoritma Genetika terdiri dari beberapa komponen utama:

- **Populasi (Population)**: Sekumpulan kandidat solusi
- **Kromosom (Chromosome)**: Representasi solusi
- **Gen (Gene)**: Bagian dari kromosom
- **Fitness Function**: Fungsi untuk menilai kualitas solusi
- **Seleksi (Selection)**: Memilih solusi terbaik untuk berkembang
- **Crossover (Rekombinasi)**: Menggabungkan dua solusi
- **Mutasi (Mutation)**: Perubahan acak untuk menjaga variasi

---

## 🔄 Alur Kerja Algoritma Genetika

1. Inisialisasi populasi secara acak
2. Evaluasi fitness setiap individu
3. Seleksi individu terbaik
4. Crossover untuk menghasilkan keturunan baru
5. Mutasi pada beberapa individu
6. Ganti populasi lama dengan populasi baru
7. Ulangi sampai kondisi berhenti terpenuhi
